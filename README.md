# 4-Bit Comparator Circuit

## Overview
This project presents the design and implementation of a 4-bit comparator circuit using basic digital logic gates. The circuit compares two 4-bit binary numbers and determines whether one number is greater than, equal to, or less than the other.

## Features
- 4-bit binary comparison
- Greater than, equal to, and less than outputs
- Implemented using basic logic gates
- Designed and simulated in Proteus

## Components Used
- Logic Gate ICs
- DIP Switch
- LED Indicators
- Resistors
- Breadboard / Virtual Simulation

## Working Principle
The circuit takes two 4-bit binary inputs and performs comparison using combinations of logic gates. Depending on the comparison result, the corresponding output becomes HIGH:
- A > B
- A = B
- A < B

LED indicators are used to display the outputs.
